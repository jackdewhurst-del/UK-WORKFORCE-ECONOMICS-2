# Workforce Economics Lab (UK)

A lightweight, static tool hub for modelling workforce risk and the value of marginal improvements.

Includes:
- Retention (churn cost + savings from small turnover improvements)
- Attraction & scaling (vacancy cost + savings from faster hiring)
- Economics (simple cost pressure / offset modelling)
- News (workforce economics headlines, best-effort RSS)
